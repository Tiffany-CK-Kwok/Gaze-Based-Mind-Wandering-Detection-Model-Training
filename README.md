# Gaze-based mind wandering detection model training
## Data Location
Data was prepared for 5 iterations of 3-fold cross-validation. Data should located in the './Data' folder. The filename systemically follows the naming pattern (training/testing)_(iteration index)_(fold index).

## Data Availability
Data privacy regulations prohibit the deposition of individual-level data to public repositories, and ethical approval does not cover the public sharing of data for unknown purposes. The minimal data set, consists of the aggregated data required to replicate the studiesâ€™ statistical findings reported, is available from the corresponding authors upon reasonable request.

## Machine Learning Models
The machine learning models were implemented in Java (SVM, Random Forest and MLP) using Weka, and in Python (XGBoost) using SKlearn.
